
### Før du starter:
Node.js og Ionic CLI må være innstallert. 

### Last ned (cmd): 
git clone https://github.com/rogerww/UUAutmat-App.git

### Installer:
Bruk *npm* fra kommandolinje for å installere alle nødvendige komponenter: 

*cd* inn i prosjektmappen.
*npm install*

### Kjør fra cmd
*ionic serve*
